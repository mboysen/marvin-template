## Afternoon / Evening (continued)

### paige eMMC cleanup — EXECUTED (was "proposed" above)
eMMC was 83% full. Root cause of the 6.1 GB `/var/log/syslog`: **logrotate was
not installed** (vestigial config, nothing running it) + the app's foreground
`docker run` sends per-frame `PIPELINE_STAGE` stdout → journald → rsyslog.
- **containerd images → NVMe** via bind-mount (`/mnt/nvme/containerd` →
  `/var/lib/containerd`, fstab `nofail`). Docker uses the **containerd
  snapshotter** (`Storage Driver: overlayfs`), so images lived on eMMC, NOT the
  NVMe Docker data-root. ~18 GB freed.
- **syslog → NVMe** (`/mnt/nvme/log/syslog` + compat symlink); installed
  logrotate 3.19.0, **30-day** stanza (daily/rotate 30/maxage 30/compress),
  timer active. Released a 6.5 GB deleted-inode a stray rsyslog held. ~6 GB freed.
- Removed stale image tags v1.2.0/v1.2.5/v1.2.9. `apt clean`.
- **Boot fail-safe**: `RequiresMountsFor=/mnt/nvme` drop-ins on docker +
  containerd (won't start on empty roots if NVMe doesn't mount).
- **Result: eMMC 83% → 39%.**
